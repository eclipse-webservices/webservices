Removing a business entity, Web service, or service interface from a registry

After you have published a business entity, Web service, or service interface, you can remove or unpublish it through the Web Services Explorer.

Prerequisites:
  1. Publish a business entity.
  2. Publish a Web service.
  3. Publish a service interface.
  4. Launch the Web Services Explorer.
To unpublish a business entity, Web service, or service interface using the Web Services Explorer:
  1. Navigate through the registry hierarchy to find the business entity, Web service, or service interface you want to unpublish. Select the business entity, Web service, or service interface.
  2. In the Actions icon bar, click the Unpublish icon.
  3. Enter the publish URL, your user ID, and password, if necessary, then click Go.

The business entity, Web service, or service interface is removed from the registry and the Web Services Explorer is automatically updated with your change.
