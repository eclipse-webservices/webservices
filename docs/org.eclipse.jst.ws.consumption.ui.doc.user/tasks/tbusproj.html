Managing referenced services

A service projection enables a business entity to reference a service that was published by another business entity. By using the businessService structure as a projection to an already published businessService, businesses can share or reuse services.  Service projections are managed centrally as part of the referencing businessEntity.

Prerequisites:
  1. Register with a registry
  2. Launch the Web Services Explorer
  3. Add the registry to the Web Services Explorer

For more information on referenced services, refer to the UDDI Version 2.0 Programmer's API Specification and the UDDI Version 2.0 Data Structure Specification.

To manage referenced services, do the following:

  1. In the Navigator pane, select the business entity for which you want to manage referenced services.
  2. In the Actions pane toolbar, click the Manage Referenced Services icon .
  3. On the Manage Referenced Services page you can add, remove, and maintain services that are referenced by the business entity.
